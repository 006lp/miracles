# 特色 Features
- 自适应设计/响应式设计
- 白昼模式和夜间模式
- 扁平化 UI 设计
- Pjax 预加载技术
- LazyLoad 图片懒加载
- 丰富的 OwO 评论表情
- 一些外部样式良好的兼容
- 说说/动态功能
- 前台登录

# 使用 Usage
1. 到 [Releases](https://github.com/BigCoke233/miracles/releases) 页面下载最新发行版。
2. Star 本项目
3. 解压后，检查文件名是否为`Miracles`，若不是请修改。
4. 根据 [Wiki](https://github.com/BigCoke233/miracles/wiki) 配置本主题
